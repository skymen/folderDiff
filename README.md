<h1 align="center">Folder Diff</h1>

Compare two folders and see what's different.

## Features

- **Visual diff tree** — Color-coded: pink (only in A), blue (only in B), yellow (different), green (match)
- **File diff viewer** — Click any changed file to see line-by-line differences
- **Change navigation** — Jump between change groups with ↑/↓ buttons
- **Collapsible folders** — Click to expand/collapse directories
- **Smart ignore** — Supports `.gitignore` files and custom glob patterns
- **Settings** — Ignore line breaks, hide matches, custom ignore patterns

## Install

```bash
npm install
npm start
```

## Build

```bash
npm run build:mac    # macOS
npm run build:win    # Windows
npm run build:linux  # Linux
```
